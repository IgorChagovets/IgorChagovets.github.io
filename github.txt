Как загрузить

1. git init
2. git add *
3. git commit -m "Message"
4. git remote add origin git@github.com:IgorChagovets/IgorChagovets.github.io.git
5. git push origin master
6. Вас спросит ваш username и password

Как обновить?

1. git add *
2. git commit -m "Я поменял название"
3. git push origin master https://oliver-twst.github.io/IgorChagovets.github.io/
4. Вас спросит ваш username и password
